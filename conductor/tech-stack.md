# Technology Stack

## Frontend
- **Framework:** React 19
- **Build Tool:** Vite 7
- **Styling:** Tailwind CSS 4
- **Mobile:** Capacitor 8

## Backend
- **Runtime:** Node.js 22
- **Framework:** Express 5

## Database & Infrastructure
- **Database:** Firebase Firestore
- **Authentication:** Firebase Authentication

## Language
- **Primary:** JavaScript (ES Modules)
